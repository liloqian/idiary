# idiary 

> 欢迎大家follow star

### idiary 是一款实用的日记记录app，有以下特点：

- 不添加任何广告。作者完全处于学习，不以商业为目的
- 可以自由添加日记分类，方便管理
- 精美的UI设计，感谢[http://www.jianshu.com/u/09ef3072029d](http://www.jianshu.com/u/09ef3072029d)的分享
- 设计有密码保护功能，保护个人隐私
- 后续网络功能正在开发中...
